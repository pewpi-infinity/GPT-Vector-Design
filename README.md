### Usage

1. **Select a Tool**: Click on any drawing tool button in the toolbar
2. **Customize Properties**: Adjust colors, stroke width, and fill settings
3. **Draw on Canvas**: Click and drag on the canvas to create shapes
4. **Ask GPT**: Type a question in the GPT Assistant box for design help
5. **Export Your Work**: Save your creation as SVG or PNG

### Keyboard Shortcuts

- `Ctrl+Z` (or `Cmd+Z` on Mac): Undo
- `Ctrl+Shift+Z` or `Ctrl+Y`: Redo

## ğŸ¯ How to Use Each Tool

### Pen Tool
- Click and drag to draw freehand
- Great for signatures, organic shapes, and sketches

### Line Tool
- Click at the starting point
- Drag to the end point and release

### Rectangle Tool
- Click at one corner
- Drag to the opposite corner and release

### Circle Tool
- Click at the center point
- Drag outward to set the radius

### Triangle Tool
- Click at the base left point
- Drag to set the width and height

### Eraser Tool
- Click and drag over areas you want to remove
- Size increases with stroke width
